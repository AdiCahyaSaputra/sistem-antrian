# Sistem Antrian

## Run On Your Local Machine
```
# Clone Repo
git clone https://github.com/Octagram82/plant-shop
cd plant-shop

# Install Dependencies
composer install
npm install

# Setup Laravel .env
cp -r .env.example .env
php artisan key:generate

# Run APP
php artisan ser
npm run dev
```
